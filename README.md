## Scrooge Project

------

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT) 

Flutter 2.10.4, Dart 2.16.2

아두이노와 통신하는 서버에서 API를 받아와 전력 복구 버튼 on/off, 전력량을 받아와 사용자에게 보여주는 역할을 합니다.

플러터를 이용하여 구현하였습니다.
